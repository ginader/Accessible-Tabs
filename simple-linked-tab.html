Usage

setting "autoAnchor" to true will make the tab widget check the url when initialized. If the url contains an anchor pointing to one of the tabs this tab will be displayed

        $(document).ready(function(){
            $(".tabs").accessibleTabs({
                tabhead:'h2',
                fx:"fadeIn",
                autoAnchor:true
            });
        });
